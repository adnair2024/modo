# Modo PreMiD Presence

This is a custom PreMiD presence for **Modo**.

## üõ†Ô∏è How to Load (Developer Mode)

Since this is a custom presence, you don't need to install it from npm. You can load it directly into your browser.

1.  **Open your Browser:** (Chrome, Edge, Firefox, etc.)
2.  **Click the PreMiD Extension Icon:** It's usually in the top-right toolbar.
3.  **Open Settings:** Click the gear/cog icon in the popup.
4.  **Enable Developer Mode:**
    *   Go to the **General** or **Developer** tab.
    *   Toggle **Developer Mode** to **ON**.
5.  **Load Unpacked Presence:**
    *   Click **"Load Presence"** or **"Select Folder"**.
    *   Select the `premid-extension` folder inside this project (`/Users/ashwinnair/Documents/GitHub/modo/premid-extension`).
6.  **Test:**
    *   Go to your [Modo App](https://p01--modo--w87x9wdr7kmw.code.run/).
    *   Check your Discord status!
